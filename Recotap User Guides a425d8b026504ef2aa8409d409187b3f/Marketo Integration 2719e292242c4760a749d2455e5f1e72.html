Marketo Authentication

Integration of Marketo and Recotap will enable one-way or two-way data sync between the two systems. In order to integrate Marketo and Recotap, please follow the steps below:

  • Login to your Recotap account, in the Client panel, navigate to the Integration screen

  • You will see a list of Apps under the CRM tab that can be integrated with Recotap. Click the Install button under the Marketo card.
  • Enter the Identity URL, Client ID & Client Secret from your Marketo account & click on Verify.

After completing the above steps, you will establish a connection between Marketo and Recotap.

Settings & Configuration

Once the Marketo Identity URL, Client ID & Client Secret is verified, the sections to configure various rules for importing and exporting data will be visible.

The rules so configured will let you import relevant Accounts, and the associated Contacts, into Recotap to create Audience segments. You can also export Recotap generated insights like engagement score and last engagement date to Marketo from Recotap.

Configuration rules highlights

  • The Settings tab hosts all the rules to import and export data From and To Marketo
  • The Field Mapping tab enables you to map individual fields between Marketo and Recotap.
  • Data from the Source system (Marketo) is imported through the Contact or Account list entity

Contact Sync

The Recotap - Marketo integration allows seamless import of Contacts. If the Contacts rules are configured, Recotap will import Marketo Contact lists to Recotap to create an Audience Segment.

New Import Rules

All the previously configured import rules for the Contacts will be listed with their source and destination details.

  • To add a new rule, click on Add Rule button which will open the configuration form.
  • To Edit an existing rule, click the 3 dots menu, the configuration form will open with the previously added rule information.
  • To search for an existing rule, enter the name of the source or rule in the search bar placed at the top-left corner of the table.

If you are adding a rule for the first time you will be taken directly to the configuration form. It allows you to add a new rule to import contacts from Marketo into Recotap. It is an inbound flow of data from Marketo to Recotap.

  • The select box Source: Marketo Contact Lists allows you to choose one or more Contact lists, created in your Marketo instance.
  • To choose a contact list, click on the Marketo contact malt-select list and choose the relevant list(s).
  • The Destination: Recotap section determines how the data imported from Marketo is processed
    • If you select the option Contacts Only, Contacts in the Marketo list are imported and created as Contacts in Recotap without them being associated with any Audience segment.
    • If you select the option New Segment, Contacts in the Marketo list are imported and will be associated with a new Audience segment. You will have to provide a name for the import process to create a new audience list, in the text field that appears below.
    • If you select the option Existing Segment, Contacts in the Marketo list are imported and associated with an existing Audience segment. You can select the relevant Audience segment from the drop-down.
  • Click on Save to add the new rule.

Edit Import Rules

You can edit existing rules by following the steps below :

  • Look for an existing rule that has to be edited from the table.
  • Identify the source list that has to be changed.
  • Click on the 3 dots menu and then select the edit option.
  • Choose the relevant destination option and its associated details
  • Click on the update button to save the changes.

Export Contact Insights

Recotap allows insights like Engagement score and last engagement date to be exported to Marketo contact. It is an outbound flow of information from Recotap to Marketo.

  • To allow this export, select the Enabled option box. This will allow the export of insights from Recotap to Marketo.
  • To prevent insight export, select Disabled.

Field Mappings

The Field mappings tab lets you map the various fields across Contacts and Accounts between the two systems. By default only mapped fields will be imported to Recotap. Hence as a one-time effort at the time of the initial configuration, the source and destination fields should be mapped. Once it's mapped click the Update button to save the mapping.
