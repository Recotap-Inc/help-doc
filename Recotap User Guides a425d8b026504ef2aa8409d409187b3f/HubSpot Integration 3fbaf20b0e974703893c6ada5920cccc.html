HubSpot Authentication

Integration of HubSpot and Recotap will enable one-way or two-way data sync between the two software. In order to integrate HubSpot and Recotap, follow the steps below:

  • Login to your Recotap account, in the Client panel, navigate to Integrations in the navigation bar.

The following screen will display all the available CRM tools that can be integrated with Recotap.

  • To choose HubSpot from the list of available CRM tools, Click on the Install button.

  • Click on Connect button to proceed and connect to HubSpot.

You will be redirected to HubSpot login page.

  • Enter your HubSpot login credentials to connect your HubSpot account with Recotap.
  • Select the account you wish to connect with Recotap.
  • To grant access to your HubSpot account, click on Connect app.

After completing the above steps, you will establish a connection between HubSpot and Recotap.

Now, HubSpot will appear in your installed CRM tools.

HubSpot Configuration

If you have the HubSpot-Recotap integration enabled, then you can import the relevant HubSpot Contacts, or existing lists into Recotap to create an Audience segment. The user can export insights like engagement score and last engagement date to HubSpot from Recotap. The user can also edit the existing segments in Recotap by appending lists from HubSpot.

  • To initiate the configuration of HubSpot, Click on the pencil icon to edit. You will see Import/Export rules under Settings. This enables you to set rules against the import and export of data between HubSpot and Recotap.
  • Field Mapping enables you to map individual fields from HubSpot to Recotap.

The import rules will display all the existing rules along with their source and destination details.

  • To Add a new rule, click on Add Rule button.
  • To Edit an existing rule, click the 3 dots Menu.
  • To search for an existing rule, enter the name of the source or rule in the search bar placed at the top-left corner of the table.

Add New Rules

With the click of the Add Rule button, a form titled Import Contact Rule will appear. It allows you to add a new rule to import contacts from Hubspot into Recotap. It is an inbound flow of data from Hubspot to Recotap.

The Source: Hubspot Contact Lists select box allows you to choose one or more Contact lists created in your Hubspot instance.

  • To choose a contact list, click on the Hubspot contact list search option and select the relevant list.
  • Select the appropriate option as the destination:
  1. If you select Contacts Only, Contacts are imported to Recotap directly without them getting associated with any Audience segment.
  1. If you select New Segment, a text field will appear below where you would need to provide a name for this list imported to Recotap to form a new segment.
  1. If you select Existing Segment, the contact list is pulled and appended to the existing segment. Click on the Existing Segment dropdown menu and select the segment to which the list is to be appended.
  • Click on Save to add the new rule.

The Export Contact Insight is the outbound flow of data where, on a periodic basis, insights like engagement score and last engagement date can be shared from Recotap to Hubspot.

  • To allow this exchange, click on the enable button. This will allow the exchange of insights from Recotap to Hubspot.
  • To prevent this exchange, click on the disabled button.

Account Sync

Hubspot integration methodology does not allow importing account sync alone. Recotap allows the insights like engagement score and last engagement date to be exchanged at the account level.

  • To allow this exchange, select enabled. This will allow the exchange of insights from Recotap to Hubspot.
  • To prevent this exchange, select disabled.

Edit Rules

The user can edit the existing rules by appending lists from HubSpot to existing segments in Recotap. Follow the steps below to edit rules:

  • Look for the existing rule that has to be edited in the import contacts section.
  • Click on the 3 dots menu and then select edit option.
  • Search the list that has to be appended in the source search option.
  • Select the destination or the existing segment from the drop down menu to which the selected list has to be appended.
  • Click on the Update button to save the changes.
