FreshSales Authentication

Integration of FreshSales and Recotap will enable one-way or two-way data sync between the two systems. In order to integrate FreshSales and Recotap, please follow the steps below:

  • Login to your Recotap account, in the Client panel, navigate to the Integration screen

  • You will see a list of CRM software that can be integrated with Recotap. Click the Install button under the FreshSales card.
  • Enter the API key & API URL from your FreshSales account & click on Verify.

After completing the above steps, you will establish a connection between FreshSales and Recotap.

Settings & Configuration

Once the FreshSales API key & API URL is verified, the sections to configure various rules for importing and exporting data will be visible.

The rules so configured will let you import relevant Accounts, and the associated Contacts, into Recotap to create Audience segments. You can also export Recotap generated insights like engagement score and last engagement date to FreshSales from Recotap.

Configuration rules highlights

  • The Settings tab hosts all the rules to import and export data From and To to FreshSales
  • The Field Mapping tab enables you to map individual fields between FreshSales and Recotap.
  • Data from the Source system (FreshSales) is imported using the Contact or Account list entity

Contact Sync

The Recotap- FreshSales integration allows to seamlessly import Contacts. If the Contacts rules are configured, Recotap will import FreshSales Contact lists to Recotap to create Audience Segment(s).

New Import Rules

All the previously configured import rules for the Contacts will be listed with their source and destination details in a tabular format.

  • To add a new rule, click on Add Rule button which will open the configuration form.
  • To Edit an existing rule, click the 3 dots menu, the configuration form will open with the previously added rule information.
  • To search for an existing rule, enter the name of the source or rule in the search bar placed at the top-left corner of the table.

If you are adding a rule for the first time you will be taken directly to the configuration form. It allows you to add a new rule to import contacts from FreshSales into Recotap. It is an inbound flow of data from FreshSales to Recotap.

  • The select box Source: FreshSales Contact Lists allows you to choose one or more Contact lists created in your FreshSales instance.
  • To choose a contact list, click on the FreshSales contact list search option and select the relevant list.
  • The Destination: Recotap section determines how the data imported from FreshSales is processed
    • If you select the option Contacts Only, Contacts in the FreshSales list are imported and created as Contacts in Recotap without them being associated with any Audience segment.
    • If you select the option New Segment, Contacts in the FreshSales list are imported and will be associated with a new Audience segment. Please provide the name of the audience in the text field that appears below.
    • If you select the option Existing Segment, Contacts in the FreshSales list are imported and associated with an existing Audience segment. Please select the relevant Audience segment from the drop-down.
  • Click on Save to add the new rule.

The Export Contact Insight is the outbound flow of data where, on a periodic basis, insights like engagement score and last engagement date can be shared from Recotap to FreshSales.

  • To allow this exchange, click on the enable button. This will allow the exchange of insights from Recotap to FreshSales.
  • To prevent this exchange, click on the disabled button.

Edit Import Rules

You can edit existing rules by following the steps below to edit rules:

  • Look for an existing rule that has to be edited from the table.
  • Click on the 3 dots menu and then select the edit option.
  • Identify the source list that has to be changed
  • Choose the relevant destination option and its associated details
  • Click on the update button to save the changes.

Export Contact Insights

Recotap allows insights like Engagement score and last engagement date to be exported to FreshSales contact. It is an outbound flow of information from Recotap to FreshSales.

  • To allow this export, select the Enabled option box. This will allow the export of insights from Recotap to Fresh Sales.
  • To prevent insight export, select disabled.

Account Sync

The Recotap - FreshSales integration allows importing of Accounts just like the Contacts. If the Account rules are enabled, Recotap can import FreshSales Account lists to Recotap to create an Audience Segment.

New Import Rules

All the previously configured import rules for the Account import will be listed with their source and destination details.

  • To add a new rule, click on Add Rule button which will open the configuration form.
  • To Edit an existing rule, click the 3 dots menu, the configuration form will open with the previously added rule information.
  • To search for an existing rule, enter the name of the source or rule in the search bar placed at the top-left corner of the table.

If you are adding a rule for the first time you will be taken directly to the configuration form. It allows you to add a new rule to import Account from FreshSales into Recotap. It is an inbound flow of data from FreshSales to Recotap.

  • The select box Source: FreshSales Accounts Lists allows you to choose one or more Account lists created in your FreshSales instance.
  • Choose relevant Account list from the multi-select list.
  • The Destination: Recotap section determines how the data imported from FreshSales is processed
    • If you select the option Accounts Only, Accounts in the FreshSales list are imported and created as Accounts in Recotap without them being associated with any Audience segment.
    • If you select the option New Segment, Accounts in the FreshSales list are imported and will be associated with a new Audience segment. You would be prompted to provide a name of the audience segment in the text field that appears below.
    • If you select the option Existing Segment, Accounts in the FreshSales list are imported and associated with an existing Audience segment. You would be prompted to select the relevant Audience segment from the drop-down to enable the Association.
  • Click on Save to add the new rule.

Field Mappings

The Field mappings tab lets you map the various fields across Contacts and Accounts between the two systems. By default only mapped fields will be imported to Recotap. Hence as a one-time effort at the time of the initial configuration, the source and destination fields should be mapped. Once it's mapped click the Update button to save the mapping.
